Cosmetic Online Business Payment Transactions Dataset

Overview:
This dataset contains transaction records from an online cosmetic business, focusing on payment activities, customer orders, and delivery details.
The dataset includes 1127 rows and 13 columns, where each row represents a unique transaction.

Dataset Columns:
Column Name  	      Description
Time & Date	      Date and time when the transaction occurred
Currency	      Currency used for the transaction (e.g., INR)
Amount	          Total transaction amount
Payment Type      Type of payment (e.g., Card, UPI, Cash)
Payment Provider  Platform used for payment (e.g., Paytm, Razorpay)
City	          Customer’s city
Order Type	      Type of order (e.g., Online, In-store)
Order ID	      Unique identifier for each order
Name	          Customer name
Quantity	      Number of items purchased
Discount	      Discount applied to the transaction (if any)
Shipping	      Shipping charges applied
Status	          Transaction status (e.g., Successful, Failed, Pending)

Purpose of the Dataset:

This dataset can be used for:
Analyzing sales performance
Understanding customer purchasing behavior
Evaluating payment method trends
Monitoring transaction success rates
Performing business intelligence and data visualization

Potential Analysis:
Most commonly used payment methods
Revenue trends over time
City-wise sales distribution
Impact of discounts on sales
Order success vs failure rates

Tools for Analysis:
Python (Pandas, NumPy, Matplotlib, Seaborn)
Microsoft Excel
Power BI / Tableau
